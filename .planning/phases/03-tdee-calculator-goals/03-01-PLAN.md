<task type="auto">
  <name>Task 2: Add TDEE calculation functions to profile service</name>
  <files>backend/src/services/profile.service.js</files>
  <read_first>
    - backend/src/services/profile.service.js (current BMI calculation pattern)
  </read_first>
  <action>
    Add two new exported functions to backend/src/services/profile.service.js:

    1. `calculateBmr(weightKg, heightCm, age, gender)` — pure function:
       - Male: 10 × weightKg + 6.25 × heightCm - 5 × age + 5
       - Female: 10 × weightKg + 6.25 × heightCm - 5 × age - 161
       - Other: use male formula
       - Return BMR as number (no rounding yet)

    2. `calculateTdee(weightKg, heightCm, age, gender, activityLevel)` — pure function:
       - Call calculateBmr() first
       - Activity multipliers: low=1.2, medium=1.55, high=1.9
       - TDEE = BMR × multiplier
       - Return rounded TDEE: Math.round(TDEE)
       - If activityLevel is null/undefined, return null

    3. `getTdeeRange(tdee)` — pure function:
       - Return { min: Math.round(tdee * 0.9), max: Math.round(tdee * 1.1) }

    4. `getCalorieTarget(tdee, fitnessGoal)` — pure function:
       - lose_weight: tdee - 500
       - maintain: tdee
       - gain_weight: tdee + 300
       - Return Math.round(result)

    5. Update `createProfile()`, `getProfile()`, and `updateProfile()` to include TDEE in return value:
       - After computing bmi/bmiCategory, also compute tdee if profile has activity_level
       - Return object becomes: { profile, bmi, bmiCategory, tdee: tdee || null, tdeeRange: tdee ? getTdeeRange(tdee) : null, calorieTarget: (tdee && profile.fitness_goal) ? getCalorieTarget(tdee, profile.fitness_goal) : null }
       - For getProfile(): use profile.activity_level and profile.fitness_goal from the database row
       - For createProfile()/updateProfile(): use the profileData.activityLevel and profileData.fitnessGoal parameters
  </action>
